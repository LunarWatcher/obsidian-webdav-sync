# Known good WebDAV servers

This folder contains documentation for known good WebDAV servers.

If your WebDAV server isn't in this list, it may still work (even without modification), but the servers in this list are known to work with this plugin, either with or without config modifications. Feel free to open a PR to add to this list if you've verified against a non-listed WebDAV server, especially if it requires extra config steps.
